SYSTEM REQUIREMENTS
-------------------
• Any text editor (Notepad, TextEdit, VS Code, Sublime)
• Any modern web browser (Chrome, Safari, Firefox)
• Internet connection (for loading Tailwind CSS from CDN)

STEP 1: UNZIP THE FILES
-----------------------
1. Extract the ZIP file to your computer
2. You should see:
   - index.html
   - README.txt
   - LICENSE.txt
   - assets/ (folder with 4 images)

STEP 2: CUSTOMIZE THE TEMPLATE
------------------------------
1. Open 'index.html' in a text editor
2. Find and replace these items:

   YOUR BRAND NAME:
   • Find: "CreatorPay™"
   • Replace: Your actual business name (appears in top left)

   YOUR PRODUCT DESCRIPTION:
   • Edit the headline text in the Hero section
   • Edit the subheadline paragraph

   YOUR PAYMENT LINK:
   • Find: "https://REPLACE-WITH-YOUR-LINK.com"
   • Replace: Your Gumroad, PayPal, or Stripe link
   • There are 3 places to replace (search for all)

   YOUR IMAGES:
   • Replace the 4 files in /assets/ folder:
     - preview-screenshot.jpg (your main product image)
     - feature-1.jpg
     - feature-2.jpg
     - feature-3.jpg
   • Keep the same file names for easiest setup
   • Recommended size: 1200x800 pixels (16:9 ratio)

   YOUR TESTIMONIALS:
   • Edit the 3 review boxes with real customer feedback
   • Change names and locations to match your audience

STEP 3: UPLOAD TO THE INTERNET
------------------------------
OPTION A - GitHub Pages (FREE):
1. Create a free account at github.com
2. Create a new repository (green button)
3. Name it: my-product-landing
4. Upload your files (drag and drop index.html + assets folder)
5. Go to Settings → Pages → Select "main branch"
6. Wait 2 minutes, your site is live!

OPTION B - Netlify (FREE):
1. Go to netlify.com
2. Drag your project folder onto the page
3. Get instant live URL
4. (Optional) Add your custom domain in settings

OPTION C - Traditional Web Hosting:
1. Use cPanel File Manager or FTP program (FileZilla)
2. Upload index.html to public_html folder
3. Upload assets folder too
4. Visit your domain to test

STEP 4: TEST EVERYTHING
-----------------------
☐ Open on your phone - should look perfect
☐ Click "Buy Now" button - goes to your payment page
☐ Click FAQ questions - they should expand
☐ Scroll through all sections - smooth animations

IMAGE TIPS
----------
• Use JPG for photos (smaller file size)
• Use PNG for graphics with transparency
• Keep images under 500KB for fast loading
• Always use 16:9 ratio (1200x800) for best fit
